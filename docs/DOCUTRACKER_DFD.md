# DocuTracker — Data Flow Diagram (DFD)

**Draw.io file (proper DFD shapes):** [`docutracker-dfd.drawio`](docutracker-dfd.drawio)

## Open in draw.io

1. Go to [app.diagrams.net](https://app.diagrams.net)
2. **File → Open from → Device**
3. Select: `c:\Users\danza\hrms_plaridel\docs\docutracker-dfd.drawio`
4. Tabs at bottom: **DFD Level 0** | **DFD Level 1**
5. Export: **File → Export as → PNG**

## Notation

| Shape | Meaning |
|-------|---------|
| Yellow rectangle | External entity (E1, E2, E3) |
| Blue circle | Process (0.0, 1.0 … 6.0) |
| Purple open rectangle | Data store (D1 … D7) |
